com.industry.analytics
Event DataSource

The com.industry.analytics.DataSource event defines a specific DataSource that is being monitored. Each DataSource can also define the relationships/grouping of different DataSources, so that they can be later referenced as a set of DataSources or as an individual DataSource. For example, an DataSource could be non-physical parent node which defines the set of sensors for a specific machine, this then has a set of physical devices associated with it.
See Also:
com.industry.analytics.SourceIdsRequest - The request event to retrieve all DataSource data associated with a specified DataSource

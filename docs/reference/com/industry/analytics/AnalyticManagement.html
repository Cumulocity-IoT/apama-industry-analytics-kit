com.industry.analytics
Event AnalyticManagement

This event exists to allow dynamic management of Analytics created using the Analytic event.

DELETE: deletes a analytic instance. RESET: resets a analytic instance to it's initial state. START: starts a analytic instance listening for incoming Data event. By default this also resets the instance. STOP: stops a analytic from listening to incoming Data events.

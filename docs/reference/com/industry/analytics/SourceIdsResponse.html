com.industry.analytics
Event SourceIdsResponse

This event is sent in response to the to the com.industry.analytics.DataSourceIdsRequest on the DataSource Service channel (I.E. "DATASOURCE_MGMT_CHANNEL") which requests the set of associated DataSources for the one specified. Note that the entire hierachy of associated child DataSources are returned in the response, not just the direct children of a specified DataSource.
See Also:
com.industry.analytics.SourceIdsRequest - The request event that this event is sent in response to.

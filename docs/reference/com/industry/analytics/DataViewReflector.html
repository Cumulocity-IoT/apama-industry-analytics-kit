com.industry.analytics
Monitor DataViewReflector

This monitor reflects Data events from the main context to the DATAVIEW_CHANNEL. This is required for the DataViewer analytic.

This overcomes the limitation of the DataView Service bundle's limitation of only routing the response events on the main context.

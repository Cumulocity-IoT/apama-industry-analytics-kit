com.industry.analytics
Event AddDataSource

The com.industry.analytics.AddDataSource event requests that a DataSource should be added to the hierachy of DataSources. This is used to define the relationships/grouping of different DataSources, so that they can be later referenced as a set of DataSources or as an individual DataSource. For example, an DataSource could be non-physical parent node which defines the set of sensors for a specific machine, this then has a set of physical devices associated with it.

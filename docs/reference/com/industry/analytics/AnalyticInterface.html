com.industry.analytics
Event AnalyticInterface

The AnalyticObject event is used to create instances of the AnalyticInterface object that can be used to manage and implement an instance of a Analytic.

This object should not be used directly, and should be created using the com.industry.analytics.AnalyticObject#init() action.

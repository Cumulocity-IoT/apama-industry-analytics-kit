Prerequisites

    The pre-requisites for the Industry Analytics Kit:

    • SoftwareAG Apama (requires v10.1 as a minimum)

SoftwareAG Designer Configuration/Setup

In order to use the Industry Analytics Kit within SoftwareAG Designer, the following steps must be taken:
  1. Run the deployment script deploy.bat – This deploys the Apama bundle configuration for the Industry Analytics Kit to SoftwareAG Designer
  2. (Re)start SoftwareAG Designer
  3. SoftwareAG Designer is now configured for you to use the Industry Analytics Kit in any projects.
    • Note: If you move the location of the Industry Analytics Kit, then you will need to re-run the deploy.bat deployment script.

Using the Industry Analytics Kit Sample

As part of the release of the Industry Analytics Kit, a sample project is provided. In order to use the sample project, the following steps must be taken:
  1. Start SoftwareAG Designer.
    • Note: Ensure that you have completed the SoftwareAG Designer configuration as described in the section above.
  2. Select File->Import from the main menu.
  3. In the Import window that appears, select General -> Existing projects into Workspace from the tree control.
  4. Select the Select root directory radio button, and click the Browse... button.
  5. Select the samples folder within the "Industry Analytics Kit Kit x.x.x.x" folder, and click the OK button.
  6. Now click Finish to import the project.
    • Note: by default, the sample will load the bundle configuration using the INDUSTRY_ANALYTICS_KIT_HOME monitorscript path variable. If you move the location of the Industry Analytics Kit, then you will need to re-run the deploy.bat deployment script.
  7. To run the sample, right-click on the name of the project, and select Run As->Apama Application.

Adding the Industry Analytics Kit to your own Apama Project

In order to use the Industry Analytics Kit in in your own Apama project, the following steps must be taken:
  1. Start SoftwareAG Designer.
    • Note: Ensure that you have configured SoftwareAG Designer as described in the section above.
  2. Either import your own pre-existing Apama project, or create a new Apama Project.
  3. Right-click on your Apama project, and select "Add Bundle", and add the "Industry Analytics Kit" from the list of bundles available. 
  4. The Industry Analytics Kit can now be used in your project.   See the Industry Analytics Kit Technical Reference document for details.

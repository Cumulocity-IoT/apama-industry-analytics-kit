Prerequisites

    The pre-requisites for the Industry Analytics Kit:

    • SoftwareAG Apama (requires v10.1 as a minimum)

Building the Industry Analytics Kit

In order to build the Industry Analytics Kit, or rebuild it after making modifications, the following steps must be taken:
  1. On a Windows platform, you can simply double-click on the "\src\build.bat" batch file.
    • Note: This batch file assumes that you have Apama installed in the default directory.  If you have installed Apama in a non-standard installation path, you will either need to modify the batch file directly in a text editor, or follow the alternate steps below.
  2. Alternatively, you can use the ANT scripts directly:
    1. Open an Apama Command Prompt.
    2. Navigate to the <analytics_kit_install_directory>/src directory.
    3. Type ant build followed by return.
      • Note: To display help on what ANT targets are available just type ant or ant usage followed by return.

Testing the Industry Analytics Kit

A test framework and a set of tests are provided to assist developing the Industry Analytics Kit further, or to provide a better understanding of the Industry Analytics Kit itself.  The test framework uses the Pysys Test Framework, based on Python,  that is shipped as part of the SoftwareAG Apama product.

Note: This section is not intended to be a guide on how the Pysys Test Framework works and how to write new tests.  Further information on Pysys can be found in the documentation for the SoftwareAG Apama product.

The tests and the Pysys Test Framework extensions can all be found in the <analytics_kit_install_directory>/tests directory.

A set of Python scripts in <analytics_kit_install_directory>/tests/src/industry/framework extend the functionality provided by Pysys to make the task of testing the Industry Analytics Kit easier.  A Pysys BaseTest is provided (see <analytics_kit_install_directory>/tests/src/industry/framework/AnalyticsBaseTest.py) which sets up the appropriate environment, starts the Correlator and injects the EPL that is required by the Industry Analytics Kit.

The individual tests themselves are under the <analytics_kit_install_directory>/tests/tests directory.  

Running the Tests

In order to run the tests, the following steps must be taken:
  1. Open an Apama Command Prompt.
  2. Navigate to the <analytics_kit_install_directory>/tests directory.
  3. Type pysys run followed by return.  This will execute all of the tests that are provided, and will indicate whether the test run passed or failed.
